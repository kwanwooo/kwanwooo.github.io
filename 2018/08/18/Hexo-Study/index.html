Hexo Study

Hexo是一个快速搭建的博客框架，支持Markdowm（或其他语言）编写文章。
安装：
依赖环境：Node.js 和 Git
在 Git Bash 输入命令 npm install -g hexo-cli
建站
在 Git Bash 中选择合适的路径，输入指令 hexo init <folder>
配置部署
在 github new repository 名字为 yourgithubname.github.io
需要将 Git 和 github SSH连接
修改 _config.yml，基本配置如下

在 Git Bash 输入 `hexo deploy`

发布（文章）
在 Git Bash 中 Hexo 文件夹下输入指令 hexo new post "articlename"
在 Hexo\source_posts 文件夹中修改 articlename.md
在 Git Bash 中输入 hexo d -g 发布成功
